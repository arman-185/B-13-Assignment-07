My project Name: KeenKeeper

=>Your personal shelf of meaningful connections. Browse, tend, and nurture the relationships that matter most.

                                      Description
KeenKeeper is a social management tool designed to help users keep track of their friendships and interactions. It provides analytics on communication habits and reminds users when it is time to reach out to friends who are "On Track", "Almost Due", or "Overdue".

Tech Stack
React.js - Frontend Framework
Tailwind CSS - Styling
React Router - Navigation
Recharts - Data Visualization (Charts)
React Toastify - Notification System
Lucide React - Icons
Core Features
Friend Dashboard & Management
View detailed profiles of your connections including status, bio, and tags.
Manage relationship goals and set "due dates" for interactions.
Interaction Logging (Quick Check-In)
Easily log interactions (Call, Text, or Video) with a single click.
Track the history of all communications with each friend.
Visual Analytics
A dynamic chart (Donut Graph) that breaks down your interaction habits.
See the distribution of Calls vs. Texts vs. Video calls to balance your communication style.
